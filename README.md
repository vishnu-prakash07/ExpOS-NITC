# ExpOS-NITC

My implementation of the eXpOS NITC roadmap. This repository tracks the journey from a bootloader to a multi-tasking OS. Currently supporting concurrent processes, kernel-level synchronization, and system call handling.

✅ Stage 1–12: Bootstrapping & XSM Basics  
✅ Stage 13–19: Interrupts, Device Handling, and Modules  
✅ Stage 20: Process Creation (Fork) & Termination (Exit)  
✅ Stage 21: Process Synchronization (Wait/Signal)
